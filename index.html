当前主模型试跑版｜不是K模型｜不是重注模型｜只输出一个主推位｜无命中直接跳

模型规则

差值定义：

D1 = A - B
D2 = B - C
D3 = C - D
D4 = D - E
D5 = E - A

主推优先级

5号特殊保护 > 6号 > 3号 > 4号

5号规则

5号进：
1）D3 = 5 且 D4 ∈ {-4, -6}
或
2）D3 ≤ -4 且 D4 = 1
过滤：若5号命中但 D5 = 0，则不主推5号。

6号规则

6号进：
1）D4 = 0
或
2）D4 < 0 且 D1 > 0 且 |D1| ≤ 4 且 |D4| ≤ 3 且 D3 ≥ 0
或
3）D1 = -2 且 D4 = -4
D5状态：D5≥0加强；D5=-1/-2正常；D5≤-3高波降级。

3号规则

3号进：
1）D2 > 0 且 D3 > 0 且 D4 < 0
或
2）D2 < 0 且 D3 ≥ 4 且 D4 < 0 且 |D4| ≤ 6
或
3）D2 < 0 且 D3 = 0 且 D4 > 0

4号规则

4号进：
1）D3 > 0 且 D4 > 0
或
2）D3 < 0 且 D4 < 0

实战纪律

只打主推；无主推跳；不打备选；5号D5=0不打；6号D5≤-3只小注观察；主推亏≤-300暂停复盘；亏≤-500模型降级；新增20-30局后重回测。
